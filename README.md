# smart-pricing-engine
Dynamic Pricing Engine for perishable goods

Overview
Smart Pricing Engine for perishable goods.
Dynamic pricing based on freshness, competitor prices, and margin floor.

Deliverables
pricer.py → CLI module.
simulation.ipynb → 7‑day simulation with charts.
sms_pricesheet.md → Business artefact.
process_log.md → Hour‑by‑hour timeline.

Installation & Usage
pip install -r requirements.txt
python pricer.py --sku TOM123 --now 2026-04-20T09:15
jupyter notebook simulation.ipynb

Business Artefact
sms_pricesheet.md is mobile‑friendly and can be shared via SMS/WhatsApp.
